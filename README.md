# Page Projet - OSINT & IA pour l'Optimisation Réseau

Interface web interactive présentant l'analyse de faisabilité d'un système prédictif pour la couverture mobile.

## Prérequis

**Python 3** doit être installé sur votre machine.

### Si Python n'est pas installé :

**Option 1 - Microsoft Store (recommandé) :**
1. Ouvrir le Microsoft Store
2. Rechercher "Python 3"
3. Cliquer sur "Obtenir" ou "Installer"

**Option 2 - Site officiel :**
1. Télécharger depuis [python.org/downloads](https://www.python.org/downloads/)
2. Exécuter l'installateur
3. Cocher "Add Python to PATH" pendant l'installation !

**Après l'installation, redémarrer la machine pour que les modifications soient effectives.**

## Lancement

Lancer powershell ou le terminal dans le fichier contenant le README

```powershell
python -m http.server 8000
```

Ouvrir http://localhost:8000 dans le navigateur.
